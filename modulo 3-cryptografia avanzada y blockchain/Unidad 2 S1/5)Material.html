Parámetros Recomendados (2024)

La criptografía caduca. Usa estas longitudes de clave mínimas:

  • Simétrico (AES): 256 bits. (128 es aceptable, pero 256 es futuro-proof).
  • Asimétrico (RSA): Mínimo 2048 bits (preferible 4096).
  • Asimétrico (ECC): Mínimo 256 bits (más eficiente que RSA).
  • Hash: SHA-256 o superior (SHA-3). Nunca MD5 o SHA-1.
Top 3 Errores de Novato
Error Por qué es peligroso
ECB Mode El modo "Electronic Codebook" cifra bloques idénticos de la misma manera. Un atacante puede ver patrones (ej. el Pingüino de Tux visible en cifrado ECB).
Hardcoding Keys Escribir la clave en el código fuente. Si subes el código a GitHub, la clave es pública para siempre.
Reutilizar IVs Usar el mismo Vector de Inicialización para dos mensajes distintos con la misma clave permite ataques de criptoanálisis.
Ejemplo Práctico: Fernet (Python)

La librería cryptography ofrece "Fernet", que implementa AES-128-CBC con HMAC y rotación de claves automáticamente. Es la forma más segura para principiantes.

from cryptography.fernet import Fernet

# Generar clave
key = Fernet.generate_key()
f = Fernet(key)

# Cifrar
token = f.encrypt(b"Mensaje secreto")
print(token)
