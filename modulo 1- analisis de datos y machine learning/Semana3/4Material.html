Guía de Código: Entrenamiento Comparativo

Usa este código base para entrenar y comparar un modelo lineal (Regresión Logística) y un modelo basado en árboles (Decision Tree). Observa cómo se generan reportes de clasificación separados para cada uno, permitiéndote ver las diferencias en métricas.

from sklearn.linear_model import LogisticRegression from sklearn.tree import DecisionTreeClassifier from sklearn.metrics import classification_report, confusion_matrix # 1. Definir Modelos # max_iter aumenta si el modelo lineal no converge. modelo_lineal = LogisticRegression(max_iter=1000, random_state=42) # max_depth limita el árbol para evitar overfitting (aprender de memoria). modelo_arbol = DecisionTreeClassifier(max_depth=5, random_state=42) # 2. Entrenar (Fit) print("Entrenando modelos...") modelo_lineal.fit(X_train, y_train) modelo_arbol.fit(X_train, y_train) # 3. Predecir y Comparar Resultados (Test Set) print("\n--- Resultados: Modelo Lineal ---") y_pred_lineal = modelo_lineal.predict(X_test) print(classification_report(y_test, y_pred_lineal)) print("\n--- Resultados: Árbol de Decisión ---") y_pred_arbol = modelo_arbol.predict(X_test) print(classification_report(y_test, y_pred_arbol))
