Recursos Complementarios

Material de profundización para reforzar la unidad de Modelado.

Datasets para Práctica de Clustering
Colección de datos sin etiqueta (Unsupervised) diseñados específicamente para practicar segmentación. Incluye datos sintéticos de comportamiento de usuarios y logs de red anónimos. Útiles para probar la capacidad de K-Means y DBSCAN para encontrar grupos ocultos.
Cómo usarlo: Intenta encontrar grupos naturales en los datos sin usar ninguna columna "target". ¿Los grupos que encuentra el algoritmo tienen sentido de negocio?
Guía Técnica: Interpretación de Curvas ROC/AUC
Lectura profunda sobre cómo interpretar la curva ROC (Receiver Operating Characteristic) y el área bajo la curva (AUC). Explica qué significa un modelo que "abraza" la esquina superior izquierda versus uno que sigue la diagonal aleatoria, y cómo usar esta métrica para comparar modelos independientemente del umbral de decisión.
Cómo usarlo: Úsalo como criterio de desempate cuando las métricas puntuales (accuracy, recall) son similares entre dos modelos.
Plantilla: Informe Ejecutivo de Selección de Modelo
Documento de texto con la estructura estándar para presentar una decisión de modelado a stakeholders no técnicos: Resumen Ejecutivo, Metodología Simplificada, Comparación de Resultados (en términos de dinero/riesgo), Análisis de Impacto y Recomendación Final.
Cómo usarlo: Copia esta estructura para tu entrega de la Actividad 3. Asegura que tu conclusión cubra tanto lo técnico como lo estratégico.
